Intellij Version

Intellij IDEA 2017.1.4

自动补全方法返回值

ctrl+alt+v

配置自动导包

setting->Editor->Auto Import 选择add unambiguous imports on the fly 配置自动导包

setting->Editor-Auto Import 选择 Optimize imports on the fly 配置去掉无用的导入包

显示类或方法的文档

ctrl+q

返回上次浏览的地方

ctrl+alt+← → 对应eclipse的alt+← →

快速打开类

ctrl+n

快速打开文件

ctrl+shift+n

显示类的方法和变量

ctrl+F12 可以输入方法名过滤

优化导入的类和包

ctrl+alt+o

跳转到方法的实现

ctrl+alt+b

跳转到大括号的开头或结尾

ctrl+[或]

##显示类结构

ctrl+h

File->setting->Editor->Appearance 选择Show whitespace;如下图

设置idea启动选择工作空间

File->setting->Appearance & Behavior 选择 System Settings 如下图

修改idea全局设置

File->other Setting->default Setting 在这里可以进行全局设置，比如设置编码、maven的配置等待

全局设置maven配置
